🛡️ Install CyberHunter Extension

  • Extract the ZIP file you just downloaded to a location on your computer
  • Open Chrome/Edge and navigate to:
    chrome://extensions
    (For Edge: edge://extensions)
  • Enable "Developer mode" using the toggle in the top-right corner
  • Click "Load unpacked" button
  • Navigate to the extracted folder and select it
  • The CyberHunter extension will now be installed and active! 🎉
⚠️ Note: Make sure the ML API server is running for full protection:
python ml-model/api_server.py
The extension will work with both Gmail and Outlook web interfaces.
