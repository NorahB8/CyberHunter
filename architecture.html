System Architecture

🎨 Frontend Layer
Web Dashboard
Single-page application for URL scanning. Built with pure HTML/CSS/JavaScript. Features real-time analysis, visual feedback, and feature breakdown.
Browser Extension
Chrome/Firefox extension with content scripts for Gmail/Outlook. Automatic email scanning, warning banners, and scan history.
🤖 Machine Learning Layer
Feature Extractor
Extracts 14+ features from URLs: length, domain age, HTTPS, suspicious characters, keywords, homograph detection, entropy, and more.
Risk Classifier
Weighted scoring algorithm that combines features to produce risk score (0-100). Classifies as high, medium, or low risk with confidence levels.
Pattern Detector
Detects phishing patterns: URL shorteners, IP addresses, suspicious TLDs, excessive subdomains, and phishing keywords.
⚡ API Layer
REST API
Flask-based REST API with endpoints for single and batch URL analysis. CORS enabled for browser integration. Returns JSON responses.
Statistics Engine
Tracks API usage, detection rates, and performance metrics. Provides real-time statistics and health monitoring.

Analysis Flow

1. URL Input
